# Frontend

> Interfaz de usuario del sistema de gestión de proyectos.

## Estado

Pendiente de implementación. Esta capa es independiente del núcleo cognitivo del agente.

## Consideraciones de Diseño

- Consumir datos desde `/app/api/`
- No contener lógica de negocio — solo presentación
- El agente opera independientemente del frontend

## Tecnologías Sugeridas

- React / Next.js (web)
- O cualquier framework compatible con la API REST de `/app/api/`
