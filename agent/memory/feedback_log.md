# Feedback Log

> Registro de feedback recibido sobre el funcionamiento del sistema, decisiones del agente o resultados de proyectos.
> Usado para mejorar el comportamiento del agente con el tiempo.

---

## Instrucciones de Uso

- Registrar feedback explícito del usuario sobre el agente o proyectos
- Incluir también lecciones de proyectos completados
- Revisar este log al inicio de nuevos proyectos similares
- Usar para ajustar `user_preferences.md` si aplica
