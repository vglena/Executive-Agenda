# Skill: Communication

## Propósito

Facilitar comunicación clara, estructurada y apropiada para cada audiencia en el contexto de gestión de proyectos.

---

## Capacidades

- Adaptar mensajes según la audiencia (equipo técnico, stakeholders, ejecutivos)
- Estructurar actualizaciones de proyecto para comunicaciones externas
- Redactar actas de reuniones
- Formular preguntas de clarificación precisas
- Sintetizar información compleja en mensajes simples
- Comunicar bloqueos y riesgos de forma apropiada

---

## Inputs Requeridos

- Información a comunicar
- Audiencia objetivo
- Formato o canal preferido

---

## Outputs

- Mensaje estructurado según audiencia
- Acta de reunión
- Actualización de proyecto para stakeholders
- Preguntas de clarificación si hay ambigüedad

---

## Herramientas Compatibles

- Cualquier canal de comunicación (el agente genera el texto)
- `directives/meeting_summary.md`
- `directives/status_report.md`

---

## Límites

- No envía mensajes directamente — genera el texto para que el usuario lo envíe
- No conoce el tono corporativo específico sin input del usuario
- No accede a correos o chats externos

---

## Principios de Comunicación Efectiva

1. **Claridad** — Un mensaje, una idea principal
2. **Contexto** — Siempre incluir por qué importa la información
3. **Acción** — Cada comunicación incluye el siguiente paso esperado
4. **Apropiado** — El nivel de detalle se ajusta a la audiencia
